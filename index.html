Why SimpleVTube

Built to get out of your way

Every decision favors one thing: getting you from download to stream as fast as possible.

Lightweight

No unnecessary features. If it doesn't help you stream faster, it isn't in the app.

Open Source

Source available on GitHub. Read the code, fork it, ship your own build — nothing is hidden.

OBS Ready

A transparent character window that drops straight into any capture source.

Easy Setup

Import two images. Select your microphone. Done — genuinely three steps.

Privacy First

No accounts. No cloud. No telemetry. Your mic never leaves your machine.

Cross Platform soon

Windows today. macOS and Linux builds are in active development.

Roadmap

Where it's going

Shipped in order, no skipped steps — each version builds on the last.

v1.0 — shipped

The foundation

  • Voice detection with auto-calibrate
  • Multi-frame idle / talking cycling
  • Transparent overlay
  • Character profiles
  • Emote hotkeys (Alt+1-9)
  • Global resize / move hotkeys
v1.1 — in progress

Feels more alive

  • Automatic blink timing
  • Better audio smoothing
  • Animated GIF support
v1.2 — planned

More control

  • Expression presets
  • Stream Deck plugin
  • macOS / Linux builds
future

Where the community takes it

  • Plugin system
  • Live2D support
  • Native OBS panel integration
